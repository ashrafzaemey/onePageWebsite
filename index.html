Things About Stellaris

Project Aim

The aim of Stellaris is to provide astronomy-related information in a manner that helps hobbyist learn and understand the night sky.

Background

Stellaris started as a hobby project. It still is a hobby project, although a rather elaborate one. We put our spare time into developing the software and aim to release a major update about once a year.

Supported OS

Stellaris requires Microsoft Windows to run. The most recent versions of Windows are supported. The current version of Stellaris runs on Windows XP, Windows Vista, and Windows 7.

StellarScript

Stellaris is able to run scripts written in a language called StellarScript. Running a script is like automating Stellaris. A script can automatically display the night sky, choose a celestial body to examine, and give explanatory comments.

Some of the Features

  • 1
  • The Educator tool interactively demonstrates and explains astronomical concepts and current phenomena.
  • 2
  • The Event Forecast automatically forecasts relevant astronomical events such as conjunctions and eclipses.
  • 3
  • Experienced users can go a step further and let Stellaris find occurrences of user-defined phenomena, like "sunrise before 7 AM".
  • 4
  • Stellaris generates reports of past and future astronomical phenomena. A report contains data for one day/night, one month, or one year.
  • 5
  • The application can calculate a wealth of data for all celestial bodies. This data can be calculated for an arbitrary time span, displayed as a graph, and exported as XML.
  • 6
  • The Tasks window suggests which actions you may want to take next. For example, Moon in the sky window, the Related Tasks offers to display an image, search for a lunar eclipse / generate a report of the Moon's ephemerides.
  • 7
  • Stellaris also features an interactive Quick Tour and an extensive help file which explain all the application's features in detail.
